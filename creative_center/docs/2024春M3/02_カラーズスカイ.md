# コンセプト
## 3 line
- 「地球のために」なんてお題目はきっと、いつだってエゴのために使われる
- 自分の意思を通さない限り、世界はずっと変わらなくて、いつだって自分自身を縛り付けたことが自分を苦しめる
- だから対抗して作るんだ。自分だけの都合で、自分のための世界を。勇気を持って！
